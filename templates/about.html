关于本项目

这是一个专为瘦子（外胚型）设计的增重增肌锻炼系统，旨在通过科学计划和持续跟踪帮助用户实现健康增肌。

系统特点

  • 个性化计划生成：利用DeepSeek AI分析用户的身体数据、可用器械、训练经验，生成适合的周锻炼计划。
  • 每日分解与跟踪：将周计划分解到每一天，提供打卡功能，记录训练感受和完成度。
  • 体重与进度可视化：记录体重变化，生成趋势图表，直观展示进步。
  • 灵活调整：根据用户反馈和体重变化，可重新生成或调整计划。
  • 全栈技术：使用Flask + PyMySQL + Bootstrap + OpenAI SDK构建，代码开源。

技术栈

后端
  • Flask - Python Web框架
  • PyMySQL - MySQL数据库驱动
  • OpenAI SDK + httpx - 调用DeepSeek API
  • python-dotenv - 环境变量管理
前端
  • Bootstrap 5 - 响应式UI
  • Chart.js - 数据可视化
  • Jinja2 - 模板引擎
  • Vanilla JavaScript - 交互逻辑

使用说明

  1. 注册账号并登录。
  2. 填写个人资料：身高、体重、目标、可用器械等。
  3. 系统生成您的第一份周锻炼计划。
  4. 每天按照计划训练，并打卡记录。
  5. 定期记录体重，查看进度图表。
  6. 根据进展调整计划或重新生成。
